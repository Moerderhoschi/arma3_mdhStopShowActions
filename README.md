MDH Stop Show Actions is a mod, created by Moerderhoschi for Arma 3. 


HOW THE ADDON WORKS?

You are able to set the showWindow config attribute for many A3 actions.


IS THE ADDON MP COMPATIBLE?

Yes, in MP only the Client/Player needs the Addon.
